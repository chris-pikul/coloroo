AlphaMapCallback

AlphaMapCallback<T>: ((alpha: number, clr: T) => number)
  • Defined in mapping.ts:34

Type Parameters

  • T extends IColor = IColor

Type declaration

    • (alpha: number, clr: T): number

    • Callback used when mapping the alpha of a color. This is separated from ChannelMapCallback to ease confusion on which index is the alpha, and because generally channel mapping happens independantly of the opacity.

      It is provided the current alpha value as a unit float (0..1), and the original color object.

      • alpha: number
