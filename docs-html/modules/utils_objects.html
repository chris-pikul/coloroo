objectMatchesPattern

  • objectMatchesPattern(obj: Record<any, any>, pattern: Record<any, any>, loose?: boolean, ignoreType?: boolean): boolean
    • Defined in utils/objects.ts:28

    Tests that all the given properties of a given object match against a pattern object.

    Each key is checked for equality, as well as the values being matched against the same type as the pattern.

    Parameters

    • obj: Record<any, any>

      Object to test

    • pattern: Record<any, any>

      Object containing dummy data to match against

    • loose: boolean = false
    • ignoreType: boolean = false

    Returns boolean

    True if all the keys and value types match against the given pattern.
